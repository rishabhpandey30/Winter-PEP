# Winter PEP – MERN Stack

This repository contains my learning during the **Winter PEP Class**, where I am learning the **MERN Stack**.

MERN stands for:
- **M**ongoDB
- **E**xpress.js
- **R**eact.js
- **N**ode.js

This is part of my hands-on learning to build full-stack web applications.

- Frontend development using React
- Backend development using Node.js and Express

- API creation and integration
